Cabeçalho O elemento "header" representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação . Um elemento "header" normalmente contém: um ou mais elementos de título
O elemento Divisão de Conteúdo. O elemento HTML "div" é o contêiner genérico para conteúdo de fluxo . Ele não tem efeito sobre o conteúdo ou layout até que seja estilizado de alguma forma usando CSS (por exemplo, o estilo é aplicado diretamente a ele, ou algum tipo de modelo de layout como Flexbox é aplicado ao seu elemento pai).
